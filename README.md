# Projeto-Dio-Beerstock-API
Projeto desenvolvido durante o bootcamp da DIO para praticar testes automatizados em uma API REST com Spring Boot. O foco está em entender fluxos, regras de negócio e validação com Mockito e MockMvc. Dentro do contexto do Breezelab, o repositório documenta o processo de aprendizado, qualidade de código e testes no dia a dia.
